# Pull Request API
 ### Для сборки проекта:

```bash
docker-compose build
```
### Для запуска проекта:
```bash
docker-compose up
```
#### P.S. О кодогенерации с помощью open api узнал только в последний момент :)
#### P.P.S. контейнер с микросервисом в редких случаях запускается не сразу, достаточно перезапустить его.
